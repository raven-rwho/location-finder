# Store Finder

another version to locate a store.

## The techstack
* expo.io
    - Used to set up the react app
* MOBX
    - state management, as it best
* react navigation
    - used for the drawer
* firebase
    - realtime database
* geofire
    - geoquery, to fetch the nearest stores and not all of them
* caroussel
    - viewpager replacement
